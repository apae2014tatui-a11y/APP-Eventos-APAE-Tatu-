
CONFIGURAÇÃO ANDROID COM FIREBASE CLOUD SYNC:

1. Firebase Setup:
   - Vá ao console do Firebase (console.firebase.google.com).
   - Crie um projeto "APAE Eventos".
   - Adicione um app Android (registre o package name ex: com.apae.eventos).
   - Baixe o arquivo 'google-services.json' e coloque em: android/app/

2. pubspec.yaml atualizado:
    dependencies:
      firebase_core: ^2.24.2
      cloud_firestore: ^4.14.0
      path_provider: ^2.1.2
      gallery_saver: ^2.3.2

3. main.dart (Inicialização):
    void main() async {
      WidgetsFlutterBinding.ensureInitialized();
      await Firebase.initializeApp();
      // Ativar persistência offline no Flutter
      FirebaseFirestore.instance.settings = const Settings(persistenceEnabled: true);
      runApp(MyApp());
    }

4. AndroidManifest.xml:
    <uses-permission android:name="android.permission.INTERNET" />
    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
